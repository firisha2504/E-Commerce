# FA Restaurant E-commerce Website

A complete, modern, and responsive e-commerce platform for FA Restaurant with delivery services, customer analytics, and admin management.

## Tech Stack

### Frontend
- **React 18** with JavaScript
- **Tailwind CSS** for responsive design
- **React Router** for navigation
- **React Query** for state management
- **Chart.js** for analytics
- **React Hook Form** for form handling

### Backend
- **Node.js** with Express.js
- **JavaScript** for server-side logic
- **JWT** for authentication
- **Bcrypt** for password hashing
- **Multer** for file uploads
- **Nodemailer** for email notifications

### Database
- **MySQL** with native MySQL2 driver
- **Redis** for caching and sessions

### Payment Integration
- Custom payment handlers for:
  - Cash on Delivery
  - Bank Transfer (CBE, Awash Bank)
  - Mobile Money (Telebirr, e-birr)

## Features

✅ Product catalog with categories (Food, Beverages)
✅ User registration and authentication
✅ Shopping cart and checkout
✅ Multiple payment methods
✅ Delivery management
✅ Customer feedback system
✅ Order history and favorites tracking
✅ Analytics dashboard
✅ Admin panel
✅ Responsive design

## Quick Start

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables
4. Run database migrations
5. Start development servers

See individual README files in each directory for detailed setup instructions.
